# stock-market-forecasting-system
Developed a cloud-based stock market forecasting system using Prophet and XGBoost, integrating sentiment analysis from multiple sources and deploying the pipeline on AWS with S3 storage and QuickSight visualization.
# Cloud-Based Stock Market Forecasting System

## Overview
This project is a cloud-based application designed to analyze and predict stock market trends using machine learning and NLP techniques.

## Technologies Used
- Python
- AWS (EC2, S3)
- Facebook Prophet
- NLP

## Features
- Real-time stock data analysis
- Sentiment analysis using NLP
- Predictive modeling for market trends
- Interactive dashboard

## Project Structure
- Source Code
- Documentation
- Presentation (PPT)

## Outcome
Provides data-driven insights to support investment decision-making.
